Explicitly Typed Expressions

Explicitly typed expressions specify a static type for an expression explicitly. For simple basic types, explicitly typed expressions result in conversions.

import ballerina/io;

public function main() {

    int intVal = 45;
    var strVal = <string>intVal;
    io:println(strVal);

Specifying string as the explicit type for an int will produce the string representation of the int.

 
    boolean boolVal = true;
    intVal = <int>boolVal;
    io:println(intVal);

Specifying int as the explicit type for boolean would result in the conversion of false as 0 and true as 1.

 
    decimal decimalVal = -10;
    boolVal = <boolean>decimalVal;
    io:println(boolVal);
}

With boolean as the explicit type for a decimal, the boolean value would be false only if the decimal value is 0.

import ballerina/io;type Person record {
    string name;
    int age;
};type Employee record {
    string name;
    int age;
    int empNo;
};
function convertEmployeeToPerson(Employee emp) {
    Person|error res = Person.convert(emp);
    if (res is Person) {
        io:println("Employee to Person, name: ", res["name"]);
    } else {
        io:println("Error occurred on conversion");
    }
}
function convertAnyMapToPerson(map<any> m) {
    Person|error res = Person.convert(m);
    if (res is Person) {
        io:println("map<any> to Person, name: ", res["name"]);
    } else {
        io:println("Error occurred on conversion");
    }
}

Type Conversion

The .convert() built-in method creates a new value with a target storage type, from a given anydata typed value, and assigns it to a target variable. Conversion attempts could be safe or unsafe, and unsafe conversion attempts could return an error. .convert() is defined on all structured types, but returns an error if the value is not of type anydata.

type Person record {
    string name;
    int age;
};

type Employee record {
    string name;
    int age;
    int empNo;
};

function convertEmployeeToPerson(Employee emp) {

Function to attempt converting an anydata record Employee, to an anydata record Person.

 
    Person|error res = Person.convert(emp);
    if (res is Person) {

Attempt creating a new value of type Person from the Employee typed emp value, without changing emp’s inherent type.

 
        io:println("Employee to Person, name: ", res["name"]);
    } else {
        io:println("Error occurred on conversion");
    }
}

If the conversion is successful, print the name field.

Function to attempt converting an any constrained map, to an anydata record Person. The conversion would return an error if non-anydata or an incompatible value is found.

 
    Person|error res = Person.convert(m);
    if (res is Person) {

        io:println("Error occurred on conversion");
    }
}

If the conversion is successful, print the name field.

Function to convert simple basic types using the .convert() built-in method.

string to int conversion is unsafe, since the string value may not be convertible to int.

 
    res1 = int.convert(s2);

    int|error intVal = int.convert(f);
    if (intVal is int) {
        io:println("int value: ", intVal);
    }

A float to int conversion can result in some of the information getting lost. However, this conversion is unsafe due to NaN or infinite float values where the conversion attempt will result in a error.

 
    boolean b = boolean.convert(s3);
    io:println("boolean value: ", b);

A string to boolean conversion is always safe. The string value true (ignoring case) evaluates to the boolean value true, while any other string is converted to the boolean value false.

 
    float|error res2 = float.convert(a);
    if (res2 is float) {
        io:println("float value: ", res2);
    } else {
        io:println("error: ", res2.detail().message);
    }
}

A simple basic typed value held in an any typed variable can also be converted to its inherent type using the .convert() method. This attempt is unsafe, since the value may not be compatible with the target type.

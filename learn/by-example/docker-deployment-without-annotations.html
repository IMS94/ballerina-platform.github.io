import ballerina/docker as _;

listener http:Listener helloWorldEP = new(9090);

service http:Service /helloWorld on helloWorldEP {
    resource function get sayHello(http:Caller caller) {
        var responseResult = caller->respond("Hello World from Docker! \n");
    }
}

Docker Deployment Without Annotations

Ballerina supports generating Docker images and the Dockerfile for services by only adding the ballerina/docker import. Docker should be configured to run the sample.

For more information, see the Docker Deployment Guide.

import ballerina/http;

import ballerina/docker as _;

Adding the import as ballerina/docker as _ will generate the Docker image and Dockerfile for the helloWorld service.

 
listener http:Listener helloWorldEP = new(9090);

service http:Service /helloWorld on helloWorldEP {
    resource function get sayHello(http:Caller caller) {
        var responseResult = caller->respond("Hello World from Docker! \n");
    }
}

# Build the Ballerina program to generate the Docker image and Dockerfile.

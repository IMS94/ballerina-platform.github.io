Preview release: Swan Lake (preview version)

If you already have jBallerina installed, you can use the Ballerina Update Tool to update it to the Swan Lake channel. To do this, first, execute the command below to get the Update Toole updated to its latest version. 

ballerina update

Next, execute the command below to update to Swan Lake Preview 1.

ballerina dist pull slp1

For further details, see the Release Note.

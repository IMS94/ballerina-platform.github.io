    var x = xml `<h:Store id = "AST" xmlns:h="http://www.test.com">
                     <h:name>Anne</h:name>
                     <h:address><h:street>Main</h:street>
                     <h:city>94</h:city></h:address>
                     <h:code><h:item>4</h:item><h:item>8</h:item></h:code>
                     </h:Store>`;
    json j1 = x.toJSON({});
    io:println(j1);
    json j2 = x.toJSON({ attributePrefix: "#", preserveNamespaces: false });
    io:println(j2);
}

XML/JSON Conversion

XML to JSON conversion is carried out via the toJSON function. This function takes the XML to be converted, and the record of options, as arguments. The options record has two fields named ‘attributePrefix’ and ‘preserveNamespaces’. The ‘attributePrefix’ field specifies the prefix added to XML attributes/namespace keys. The default value is ‘@’. The ‘preserveNamespaces’ field specifies whether to preserve the namespace prefixes during the conversion.

This creates an XML and associates it with a variable.

 
    json j1 = x.toJSON({});
    io:println(j1);

This converts the XML to JSON with namespaces and the default attributePrefix.

 
    json j2 = x.toJSON({ attributePrefix: "#", preserveNamespaces: false });
    io:println(j2);
}

This converts the XML to JSON without namespaces and with a custom attributePrefix.

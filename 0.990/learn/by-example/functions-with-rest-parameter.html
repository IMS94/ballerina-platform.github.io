import ballerina/io;
function printFruits(string separator,
                     string title = "Fruits: ",
                     string... concatStrings) {    string finalStr = "";
    int index = 0;
    foreach var str in concatStrings {
        if (index == 0) {
            finalStr = str;
        } else {
            finalStr = finalStr + separator + str;
        }
        index += 1;
    }    io:println(title + finalStr);
}public function main() {
    printFruits(",");
    printFruits(",", "Apples");
    printFruits(",", title = "Available Fruits: ", "Apples");
    printFruits(",", "Apples", "Oranges");
    printFruits(",", title = "Available Fruits: ", "Apples", "Oranges",

    printFruits(",", "Apples", "Oranges", title = "Available Fruits: ",

    string[] fruits = ["Apples", "Oranges", "Grapes"];
    printFruits(",", title = "Available Fruits: ", ...fruits);
}

Rest Parameters

A Ballerina function can have at most one rest parameter, and the rest parameter can take zero or more values. The rest parameter inside a function is equivalent to an array of the same type.

import ballerina/io;

function printFruits(string separator,
                     string title = "Fruits: ",
                     string... concatStrings) {

This function takes one required parameter, one defaultable parameter and one rest parameter of type string. A function can have only one rest parameter. The rest parameter can take any number of values, and is equivalent to an array of the same type.

 
    string finalStr = "";
    int index = 0;
    foreach var str in concatStrings {
        if (index == 0) {
            finalStr = str;
        } else {
            finalStr = finalStr + separator + str;
        }
        index += 1;
    }

    io:println(title + finalStr);
}

    printFruits(",");

Call the function by passing only the required parameter.

 
    printFruits(",", "Apples");

Call the function by passing the required parameter, and one value for the rest parameter.

 
    printFruits(",", title = "Available Fruits: ", "Apples");

Call the function by passing the required parameter, defaultable parameter, and one value for the rest parameter.

 
    printFruits(",", "Apples", "Oranges");
    printFruits(",", title = "Available Fruits: ", "Apples", "Oranges",
                "Grapes");

Call the function by passing a separator, and multiple values for the rest parameter.

 
    printFruits(",", "Apples", "Oranges", title = "Available Fruits: ",
                "Grapes");

The placement of defaultable parameters can be mixed with rest parameters when invoking the function.

 
    string[] fruits = ["Apples", "Oranges", "Grapes"];
    printFruits(",", title = "Available Fruits: ", ...fruits);
}

Pass an array as the rest parameter instead of calling the function by passing each value separately.

import ballerina/io;type Person record {
    string name = "";
    int age = 0;
};type Employee record {
    string name = "";
    int age = 0;
    int empNo = 0;
};function assertTypes(anydata emp) {
    Employee employee = <Employee>emp;
    io:println("Type asserted employee name: ", employee.name);
    Person|error person = trap <Person>emp;
    io:println("Type asserted person name or error: ",
               (person is Person) ? person.name : person.detail().message);
}public function main() {
    Employee emp = { name: "Speedy Gonzales", age: 4, empNo: 1 };
    assertTypes(emp);
}

Type Assertion

Type assertion is a check for exact type. Type assertion checks if a given value’s storage type is exactly a particular type. Type assertion results in panic, if assertion fails (i. e., the value is not of the asserted type).

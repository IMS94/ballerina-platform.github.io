Arrays

In Ballerina, arrays are mutable lists of values with dynamic length, where each member in the list is of the same type. The implicit initial value of an array is the array with a length of 0. If you want to have a fixed length array, specify the length of the array at the time of declaration.

This creates an int array of length 0.

 
    int[] b = [1, 2, 3, 4, 5, 6, 7, 8];
    io:println(b[0]);
    io:println(b.length());

This assigns an array literal.

 
    b[999] = 23;
    io:println(b[999]);
    io:println(b.length());

Arrays are unbounded in length. They can grow up to any length based on the given index. In this example, the length of the array is 1000.

 
    int[][] iarray = [[1, 2, 3], [10, 20, 30], [5, 6, 7]];
    io:println(iarray.length());
    io:println(iarray[0].length());

This initializes an array of int arrays.

 
    iarray = [];
    int[] d = [9];
    iarray[0] = d;

This initializes the outermost array with the implicit default value.

 
    io:println(iarray[0][0]);

This prints the first value of the two-dimensional array.

 
    int[5] e = [1, 2, 3, 4, 5];
    io:println(e.length());

This creates an int array with a fixed length of five.

 
    int[!...] g = [1, 2, 3, 4];
    io:println(g.length());
}
